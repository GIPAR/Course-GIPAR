    02. Configuration Models Gazebo

Configure Models

In this topic you will learn how to manually configure gazebo models.

  • 1: For the gazebo simulations you will need models, however they are synced with the server and this procedure may take a while, we recommend downloading them manually
    • 1.1: First go to the website and click on the GREEN BUTTON with the code name, then choose the DOWNLOAD ZIP option
    • 1.2: Now go to your HOME folder and press CTRL H keys, this will make hidden folders to be revealed, every hidden folder has a dot at the beginning of the name, see the picture:

      Enter the .gazebo folder, check if the models folder exists, if not, cut the file you downloaded gazebo_models-main.zip, extract the contents, rename the folder gazebo_models-main to models and that's it. The result should look like the image below:


      To confirm that everything went well, enter the models folder and it should contain all gazebo models
